# SKULL DOWNLOADER
![SkullDownloaderLogo](https://raw.githubusercontent.com/dogaegeozden/skull_downloader/main/logos/skull_downloader_logo.jpg)

## INSTALLATION
1) ```curl -L https://github.com/dogaegeozden/skull_downloader/releases/download/Download/skull-downloader.deb -o skull-downloader.deb```
2) ```sudo dpkg -i skull-downloader.deb```

## HELP

Skull downloader is downloading media from YouTube using pytube package. And, there is a problem with download it may turn back to normal couple days later.
